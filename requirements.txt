Reproducibility requirements

This codebase was verified in the following environment.

Operating system: Windows 11 x64
R: 4.4.3
Python: 3.12.7
Java: OpenJDK 17.0.15
River: 0.23.0

Java is required for rJava and RMOA.
Python is required for reticulate and the River-based baselines.
The file python/river_helpers.py must be available for experiments using forwardRiver().

Required R packages can be installed in R with the following command.

install.packages(c(
“RMOA”, “rJava”, “reticulate”, “sgd”,
“dplyr”, “ggplot2”, “zoo”, “patchwork”, “readr”, “scales”,
“latex2exp”, “cowplot”, “tidyr”, “future.apply”, “dynaTree”,
“colorspace”, “gridExtra”, “ggforce”
))
